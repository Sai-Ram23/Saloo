# Saloo - Salon Booking & Discovery Web Application

**Saloo** is a modern and elegant web application designed to help users discover and book top-rated salon services. Whether it's a haircut, facial, makeup, or body treatment, Saloo connects customers with professional stylists and beauty experts.

*   **Pricing Plans**: Clear display of service costs and packages.
*   **Gallery**: Showcase of hairstyles, makeup looks, and salon ambience.
*   **Team Profiles**: Meet the stylists, directors, and makeup artists.
*   **Testimonials**: Real reviews from happy customers.
*   **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
*   **Blog Section**: Latest news, beauty tips, and trends.
